# ionic-server

Baixe o repositório
```
https://github.com/nicholasess/ionic-server
```

Instale as dependências
```
npm install
```

Execute o server
```
node bin/www
```
Para acessar a api no browser
```
http://localhost:3000
```

Rotas do usuário:
```
POST /api/signin
POST /api/signup
```
Rotas da lista
```
GET /api/listas
POST /api/listas

GET /api/lista/:id
PUT /api/lista/:id
DELETE /api/lista/:id
```
